A1.18 第十六类数据格式：预报站点数据

 

文件头：

diamond 16 数据说明（字符串） 总站点数（整数）

 数据：

站点标识（长整数，一般为区站号）  纬度  经度  级别（均为整数）

注：纬度和经度均为60进制，并乘以100后化为整数。

 

例子：

diamond 16 stations 400

50136   5328  12222  0

50246   5219  12443  0

50353   5143  12639  0

50434   5029  12141  0

50442   5024  12407  0

50468   5015  12727  0

50527   4913  11945  1

50548   4912  12343  0

......
